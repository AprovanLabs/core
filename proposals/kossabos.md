# Kossabos

## Overview

Consistent design and API for small, New York Times-like games. Create and share games with built-in multiplayer and social features.

## Motivation

The New York Times has a number of small games that are fun and engaging. They have a clean, consistent design. Kossabos would implement a strict system for enforcing a similar feel while providing re-usable components for similar pieces of functionality.

This would allow any user to create and distribute new, engaging games that can be enjoyed by friend-groups looking for daily challenges and brain-teasets, a la Wordle
